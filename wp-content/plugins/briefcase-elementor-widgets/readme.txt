== Description ==

Add some new widgets to the popular free page builder Elementor.

== Installation ==

1. Upload `briefcase elementor widgets` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
